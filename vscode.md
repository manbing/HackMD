# Keyboard ShortCuts
⌥ is Option / ALT
^ is Control
← is Left arrow
→ is Right arrow
⇧ is Shift


## Code Navigation


| Action     |     Windows     |     MAC     |
| -------- | -------- | -------- |
| Go Forward     |     Ctrl + Click         |    Ctrl + Click       |
| Go Forward     |   ALT + RightArrow     |   ⌥ + RightArrow     |
| Go Back     |     ALT + ←         |   ⌥ + ←            |
| Jump To Bracket     |    Ctrl + Shift + \ |   Cmd + Shift + \  |
